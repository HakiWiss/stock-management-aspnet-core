# 📦 Stock Management System - ASP.NET Core MVC

## 📖 Description
This is a web-based stock management system developed using ASP.NET Core MVC.  
It helps small business owners manage products, suppliers, categories, and stock movements efficiently.

---

## 🚀 Features

- Product management (CRUD)
- Supplier management
- Category organization
- Stock entry & exit tracking
- Real-time stock updates
- Data validation (price, quantity, etc.)

---

## 🛠️ Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- Razor Views

---

## 🧱 Architecture

The project follows the MVC pattern:
- Models: Business logic & data
- Views: UI with Razor
- Controllers: Handle requests & logic

---

## 🗄️ Database

Main entities:
- Product
- Supplier
- Category
- StockMovement
